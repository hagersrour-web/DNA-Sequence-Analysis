# DNA Sequence Analysis

A simple Bioinformatics project written in Python for analyzing DNA sequences.

## Features

- Count total nucleotides
- Calculate GC Content (%)
- Count Purines (A and G)
- Count Pyrimidines (C and T)
- Calculate Purine Content (%)

## Example

Input:

AGTTAGCTAGGAG

Output:

Nucleotides are 13
GC Content is 46.15 %
Purines are 9
Pyrimidines are 4
Purines Content is 69.23 %

## Technologies Used

- Python
- Basic Bioinformatics
- String Manipulation
